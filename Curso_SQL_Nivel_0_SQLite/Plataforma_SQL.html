    Ayuda rapida del laboratorio

    Resumen de uso sin ocupar espacio fijo en pantalla.
    1. Selecciona una unidad del menu izquierdo. Puedes ocultar el menu y recuperarlo con el boton flotante.
    2. Abre teoria, ejercicio o solucion. Si seleccionas un bloque del contenido y pulsas Pasar bloque al editor, solo se copia ese fragmento.
    3. En el editor puedes ejecutar solo la seleccion actual con Ctrl+Enter o con Ejecutar seleccion.
    4. Las tarjetas superiores de socios, libros y prestamos cargan una consulta util para inspeccionar esa tabla.
    5. Si quieres más espacio, usa Ocultar tablas, Ocultar teoria u Ocultar laboratorio.
    6. Reset BD restaura la base Biblioteca Nivel 0 en memoria al estado original del curso.
